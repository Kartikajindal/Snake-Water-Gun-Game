# Snake-Water-Gun-Game

A simple command-line based Snake Water Gun game built using Python where the user plays against the computer. The computer randomly selects its move using Python’s random module, and the winner is decided based on game logic implemented using dictionary mapping and arithmetic comparison.


## 🚀 Features


-CLI-based interactive gameplay

-Random computer move generation

-User vs Computer gameplay

-Dictionary-based choice mapping

-Mathematical logic to determine winner

-Win / Lose / Draw result display


## 🧩 Game Rules


-Snake drinks Water → Snake Wins

-Water damages Gun → Water Wins

-Gun kills Snake → Gun Wins

-Same choices → Draw


## 🛠️ Technologies Used


-Python 3

-Random Module

-Dictionaries

-Conditional Statements

-Command Line Interface (CLI)

## ▶️ How to Run the Game


-Make sure Python is installed on your system.

-Clone this repository or download the project files.

-Open terminal or command prompt in the project folder.

-Run the following command:

 python snake_water_gun.py
